API Documentation

Get /todos

This route will return an array for each todo item in our app. But only the id, task, and is_Done are included.

POST /todos

This route can be used to create a todo item. The body of the request should include the task and description. { "task": "feed the dog", "description": "feed the dog 2 scoops of dry food." }
